# Balloon Blast Range

A fast, arcade-style balloon shooter built with HTML5 Canvas. Aim, shoot, and pop balloons before they escape. The game includes power-ups, sound effects, difficulty modes, key rebinding, and mobile controls.

## Features
- Mouse/touch aiming with click/tap shooting
- Keyboard aiming and movement (rebindable)
- Power-ups: Multi Shot, Slow Time, Piercing Shots, Score x2
- Difficulty modes (Easy / Normal / Hard)
- High score saved in localStorage
- Sound effects with toggle
- Mobile on-screen controls
- Pause menu with key rebinding

## Controls (Default)
- Move: A / D
- Aim: Arrow Left / Arrow Right
- Shoot: Space
- Pause: P

You can rebind keys in the pause menu.

## How to Run
1. Open `index.html` in a browser.
2. Choose difficulty and sound settings.
3. Click **Start Game**.

## Tips
- Use power-ups to survive higher levels.
- Piercing Shots can pop multiple balloons per shot.
- Score x2 doubles points while active.

## Tech
- HTML5 Canvas
- CSS
- Vanilla JavaScript

Enjoy the pop!
